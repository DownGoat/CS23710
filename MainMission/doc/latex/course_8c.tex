This function reads and parses the file with the courses. 

The function will check that the node specified exists, and that there is a valid track between the nodes. The data should consist of a single capital character which will be the course I\-D. Followed by a positive integer which will be the number of nodes for the course. The rest of the line should be the I\-D of each node separated by spaces.


\begin{DoxyParams}{Parameters}
{\em path} & Path to the file containing the course data.\\
\hline
{\em \hyperlink{structnode__list}{node\-\_\-list}} & Pointer to the node list. It is needed to verify that the nodes in the course exists.\\
\hline
{\em \hyperlink{structtrack__list}{track\-\_\-list}} & Pointer to the track list. It is needed to verify that there is a valid path between the nodes of the course.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
If the function is successful it will return a pointer to the first element of the list. If it fails it will return N\-U\-L\-L.
