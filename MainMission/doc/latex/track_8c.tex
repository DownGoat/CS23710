This function reads in and parses a file containing the track data. 

It uses this data to create a linked list of the tracks. The file should have data about paths between nodes and the maximum time a entrant should use between the nodes.


\begin{DoxyParams}{Parameters}
{\em path} & The path to the file containing the track data.\\
\hline
{\em \hyperlink{structnode__list}{node\-\_\-list}} & Pointer to the list of nodes, the list is needed to look up the nodes that are used in the track data to verify that the track is between nodes that exist.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Upon success this function returns a pointer to the first element in the linked list. If it fails it will return N\-U\-L\-L.
